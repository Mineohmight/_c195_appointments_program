Purpose: To develop a GUI-based desktop scheduling application 
for a global consulting organization.

Author: Riley O'Donnell


Email: rodon15@wgu.edu

Version: 1.0.0

How to run the program: 
- Program was created using NetBeans
- To run in NetBeans:
	- Create a new NetBeans project
	- Create packages (under "Source Packages"):
		- "controller"
		- "database"
		- "main"
	- Create packages (under "Other Sources"):
		- "main"
	- Import files into packages, referring to folder 		  structure of project
	- .fxml files go in "Other Sources" > "main"
	- .properties files go in "Other Sources" > "<default             	package>"
	- module-info.java goes in "Source Packages" > 	"<default package>"
	- replace pom.xml with pom.xml from project

Additional Report: User report, a report which shows the appointments that are attached to a specific user, either user "test" or user "admin".
